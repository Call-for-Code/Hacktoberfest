# ClusterDuck Protocol

## Description

This mobile mesh network of IoT devices based on LoRa can be accessed by people to stay in touch when communication lines go down following a disaster.

## Technology stack

- IBM Watson
- IBM IoT Platform
- Apache-2.0
- C++
- Java
- JavaScript
